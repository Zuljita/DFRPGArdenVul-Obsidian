# Locations

- Create one file per location in this folder.
- Use the template in `vault/templates/Location Template.md`.
- Name files in Title Case (e.g., `Pyramid of [[npcs/Thoth.md|Thoth]].md`).

Suggested front matter:
---
title: Pyramid of [[npcs/Thoth.md|Thoth]]
tags:
  - location
  - location
aliases: []
---
